# Support

For questions and bug reports, use GitHub Issues.

## Before Opening an Issue

Please check:

- the latest release is installed
- the VM can start
- the project appears in the Projects list
- `/etc/hosts` contains the JimmyBox block
- the selected PHP version matches the project requirements

## Useful Diagnostics

Include these details when possible:

- macOS version
- Mac model and Apple Silicon generation
- JimmyBox Studio version
- project domain
- CMS adapter
- PHP version
- the exact error message shown by the app

Do not paste secrets, tokens, private database dumps, or customer data into a public issue.
