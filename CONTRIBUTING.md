# Contributing

Thank you for taking the time to improve JimmyBox Studio.

## Before You Start

This repository is public, but it is not licensed as open source. By opening an
issue or pull request, you agree that your contribution can be reviewed and, if
accepted, incorporated into JimmyBox Studio. You do not receive rights to reuse
or redistribute the project outside this repository.

## Useful Commands

```sh
npm install
npm test
npm start
```

## Pull Request Checklist

- Keep changes focused.
- Avoid unrelated formatting churn.
- Add or update tests for behavior changes.
- Run `npm test` before opening a pull request.
- Do not commit credentials, local project archives, built DMGs, `node_modules`, or VM assets.

## Reporting Issues

Please include:

- macOS version and Mac model.
- JimmyBox Studio version.
- What you expected to happen.
- What happened instead.
- Relevant project type, PHP version, and CMS.
- Logs or screenshots when they help.
